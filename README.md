# 🤖 Artificial Intelligence (Machine Learning & Deep Learning)

**Government of Pakistan – NAVTTC**
**Prime Minister's Hunarmand Pakistan Program – "Skills for All"**

---

## 📌 Course Overview

This is a **3-month intensive Artificial Intelligence program** focused on building industry-ready skills in:

- Machine Learning (ML)
- Deep Learning (DL)
- Natural Language Processing (NLP)
- Microsoft Azure AI Services

> The course emphasizes **hands-on learning (80%)** with real-world problem-solving and deployment experience.

---

## 🎯 Objectives

- Develop employable AI & ML skills
- Gain expertise in NLP specialization
- Learn deployment using Microsoft Azure AI
- Build real-world AI solutions
- Prepare for industry roles in AI

---

## 🧠 Learning Outcomes

By the end of this course, you will:

- ✅ Understand core AI & ML concepts
- ✅ Apply state-of-the-art ML techniques
- ✅ Perform Exploratory Data Analysis (EDA)
- ✅ Design, train & evaluate models
- ✅ Work with:
  - `scikit-learn`
  - `NumPy`, `Pandas`
  - `TensorFlow`, `PyTorch`, `Keras`
- ✅ Build NLP systems:
  - Text preprocessing (stemming, lemmatization)
  - POS tagging, NER
  - Word embeddings & deep learning models
- ✅ Deploy AI solutions using Microsoft Azure

## 📅 Course Structure (12 Weeks)

### 🔹 Week 1–2: Foundations
- AI Introduction
- Linux Commands
- Python Basics (OOP, Functions, Loops)

### 🔹 Week 3–4: Data Analysis
- Statistics & Probability
- NumPy & Pandas
- Data Visualization

### 🔹 Week 5: Machine Learning I
- Linear Regression
- Polynomial Regression
- Logistic Regression

### 🔹 Week 6: Machine Learning II + NLP
- SVM, Decision Trees, Random Forest
- NLP Basics (Tokenization, POS, NER)

### 🔹 Week 7–9: Deep Learning
- Neural Networks (MLP)
- CNN (Image & Text)
- RNN, LSTM, GRU
- Word Embeddings (Word2Vec)

### 🔹 Week 10: Advanced Topics
- Attention Mechanism
- Sequence Models
- Project Planning

### 🔹 Week 11–12: Azure AI + Final Project
- Azure AI Services
- NLP & Vision APIs
- Deployment & CI/CD
- Final Project + Viva

## 📂 Hands-on Tasks

**25 task files** currently covering Python fundamentals, with more ML/DL/NLP tasks to be added as the course progresses:

| Tasks | Topics |
|-------|--------|
| Task 1 | Environment setup (Anaconda, PyTorch, TensorFlow, VS Code, PyCharm) |
| Tasks 3–10 | Basic Python: arithmetic, math operations, unit conversions |
| Tasks 11–14 | Conditional statements (if, if-else, nested if, elif) |
| Tasks 15–19 | Loops: `for`, `while`, `break`, `continue`, `enumerate` |
| Task 20 | Functions: parameters, return values |
| Tasks 21–25 | Data structures: List, Tuple, String, Set, Dictionary |
| Tasks 26–27 | Exception handling and custom exceptions |

> 📌 NumPy & Pandas, Machine Learning, Deep Learning, and NLP tasks will be added in subsequent weeks.
